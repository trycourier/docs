# Audit Trail

All Courier business tier customers gain access to a workspace audit trail which allows you to track a wide range of user activity for all members in a workspace. The audit trail will display supported events related to workspace authentication, security, and core functionality when triggered.

## Supported Events

### API Key

- API Key Created
- API Key Deleted
- API Key Rotated

### Automation Template

- Automation Template Published
- Automation Template Deleted

### Brand

- Brand Created
- Brand Updated
- Brand Default Brand Updated
- Brand Published
- Brand Deleted

### Notification

- Notification Draft Created
- Notification Duplicated
- Notification Rolled Back
- Notification Subscription Topic Changed
- Notification Published
- Notification Deleted

### Preferences

- Preferences Page Published
- Preferences Notification Linked
- Preferences Notification Unlinked
- Preferences Section Created
- Preferences Section Deleted
- Preferences Section Channel Added
- Preferences Section Channel Removed
- Preferences Topic Created
- Preferences Topic Deleted
- Preferences Topic Default Status Changed

### User

- User Invited
- User Role Changed
- User Deleted
- User Logout

### Workspace

- Workspace Name Changed
- Workspace Discoverability Enabled
- Workspace Discoverability Disabled
- Workspace Accessibility Changed
- SSO Required Enabled
- SSO Required Disabled
- Click-Through Tracking Enabled
- Click-Through Tracking Disabled
- Email Open Tracking Enabled
- Email Open Tracking Disabled
- Send Limits Settings Updated

- Segment Source Restored
