# Delete an Account

**URL:** `https://api.courier.com/accounts/:account_id`

**Method:** `DELETE`

**Path Parameters:**

| Key        | Type   |
| ---------- | ------ |
| account_id | string |

**Response:**

Status: 202

**Example:**

```js
DELETE https://api.courier.com/accounts/ACME
```
