# Delete an Account

**URL:** `https://api.courier.com/accounts/:account_id`

**Method:** `DELETE`

**Path Parameters:**

<Params>
  <Param name="account_id" type="string" required>
    Id of the account to be deleted.
  </Param>
</Params>

**Response:**

Status: 204

**Example:**

```js
DELETE https://api.courier.com/accounts/ACME
```
