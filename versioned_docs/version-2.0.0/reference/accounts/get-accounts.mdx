# Get a List of Accounts

**URL:** `https://api.courier.com/accounts`

**Method:** `GET`

**Path Parameters:**

None

**Query Parameters:**

| Key            | Type   | Required | Default      | Description                               |
| -------------- | ------ | -------- | ------------ | ----------------------------------------- |
| limit          | number | No       | 20 (max 100) |                                           |
| starting_after | string | no       |              | Value of next_page from previous response |

**Response Body:**

| Field     | Type                | Description                                          |
| --------- | ------------------- | ---------------------------------------------------- |
| items     | Account[]           | An array of account objects                          |
| has_more  | boolean             |                                                      |
| next_page | string \| undefined | A cursor to provide for the starting_after parameter |

**Example:**

```js
GET https://api.courier.com/accounts

RESPONSE BODY
{
  items: [
    {
      id: "ACME",
      name: "ACME Inc",
      properties: {
        ceo: "Person",
      },
      user_profile: {
        locale: "en-US",
        "slack": {
          "access_token": "xoxb-bunch-o-letters",
        }
      }
    }
  ],
  has_more: false
}
```
