# Create or Replace an Account

**URL:** `https://api.courier.com/accounts/:account_id`

**Method:** `PUT`

**Path Parameters:**

| Key        | Type   |
| ---------- | ------ |
| account_id | string |

**Body:**

| Field               | Type   | Optional | Default | Description                                                                                                                                          |
| ------------------- | ------ | -------- | ------- | ---------------------------------------------------------------------------------------------------------------------------------------------------- |
| name                | string | no       |         |                                                                                                                                                      |
| parent_account_id   | string | yes      |         |                                                                                                                                                      |
| default_preferences | object | yes      |         | Account level default_preferences object                                                                                                             |
| properties          | object | yes      |         | arbitrary properties accessible from the template                                                                                                    |
| user_profile        | object | yes      |         | a base profile object merged with user profile on supplied on send                                                                                   |
| brand_id            | string | yes      |         | When specified, this brand will be used over the default workspace brand. A brand specified directly in the send call still takes highest precedence |

**Response Body**

A copy of the created account.

| Field               | Type   | Description                                            |
| ------------------- | ------ | ------------------------------------------------------ |
| id                  | string | Id of the account                                      |
| name                | string |                                                        |
| parent_account_id   | string |                                                        |
| default_preferences | object | Account level default_preferences object               |
| properties          | object | Arbitrary properties accessible from the template      |
| user_profile        | object | A user profile object merged with user profile on send |
| brand_id            | string |                                                        |

default_preferences object can include following attributes:

| Field | Type  | Optional | Description                                            |
| ----- | ----- | -------- | ------------------------------------------------------ |
| items | array | no       | Array of subscription topic objects with id and status |

default_preferences topics object can include following attributes:

| Field  | Type   | Optional | Description                                              |
| ------ | ------ | -------- | -------------------------------------------------------- |
| id     | string | no       | Topic ID                                                 |
| status | string | no       | Possible values are "OPTED_OUT", "OPTED_IN" or "REQUIRED |

**Example:**

```js
// PUT https://api.courier.com/accounts/ACME

// BODY
{
  name: "ACME Inc",
  brand_id: "acme_brand",
  properties: {
    ceo: "Person",
  },
  default_preferences: {
    items: [{
      id: "my-topic",
      status: "OPTED_IN"
    }]
  },
  user_profile: {
    locale: "en-US",
    "slack": {
      "access_token": "xoxb-bunch-o-letters",
    }
  }
}

// Response Body
{
  id: "ACME",
  name: "ACME Inc",
  brand_id: "acme_brand",
  properties: {
    ceo: "Person",
  },
  default_preferences: {
    items: [{
      id: "my-topic",
      status: "OPTED_IN",
      type: "subscription_topic"
    }]
  },
  user_profile: {
    locale: "en-US",
    "slack": {
      "access_token": "xoxb-bunch-o-letters",
    }
  }
}
```
