# Create an Account

**URL:** `https://api.courier.com/accounts/:account_id`
**Method:** `POST`

**Path Parameters:**

| Key        | Type   |
| ---------- | ------ |
| account_id | string |

**Body:**

| Field             | Type   | Description | Required | Default |
| ----------------- | ------ | ----------- | -------- | ------- |
| name              | string |             | yes      |         |
| parent_account_id | string |             | no       |         |
