# Create or Replace an Account

**URL:** `https://api.courier.com/accounts/:account_id`

**Method:** `PUT`

**Path Parameters:**

| Key        | Type   |
| ---------- | ------ |
| account_id | string |

**Body:**

| Field             | Type   | Required | Default | Description                                                                                                                                          |
| ----------------- | ------ | -------- | ------- | ---------------------------------------------------------------------------------------------------------------------------------------------------- |
| brand_id          | string | no       |         | When specified, this brand will be used over the default workspace brand. A brand specified directly in the send call still takes highest precedence |
| name              | string | yes      |         |                                                                                                                                                      |
| parent_account_id | string | no       |         |                                                                                                                                                      |
| properties        | object | no       |         | arbitrary properties accessible from the template                                                                                                    |
| user_profile      | object | no       |         | a base profile object merged with user profile on supplied on send                                                                                   |

**Response Body**

A copy of the created account.

| Field             | Type   | Description                                            |
| ----------------- | ------ | ------------------------------------------------------ |
| brand_id          | string |                                                        |
| id                | string | Id of the account                                      |
| name              | string |                                                        |
| parent_account_id | string |                                                        |
| properties        | object | Arbitrary properties accessible from the template      |
| user_profile      | object | A user profile object merged with user profile on send |

**Example:**

```js
// PUT https://api.courier.com/accounts/ACME

// BODY
{
  name: "ACME Inc",
  brand_id: "acme_brand",
  properties: {
    ceo: "Person",
  },
  user_profile: {
    locale: "en-US",
    "slack": {
      "access_token": "xoxb-bunch-o-letters",
    }
  }
}

// Response Body
{
  id: "ACME",
  name: "ACME Inc",
  brand_id: "acme_brand",
  properties: {
    ceo: "Person",
  },
  user_profile: {
    locale: "en-US",
    "slack": {
      "access_token": "xoxb-bunch-o-letters",
    }
  }
}
```
