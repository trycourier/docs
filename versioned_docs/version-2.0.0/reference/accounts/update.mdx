# Update an Account

**URL:** `https://api.courier.com/accounts/:account_id`

**Method:** `POST`

**Path Parameters:**

| Key        | Type   |
| ---------- | ------ |
| account_id | string |

**Body:**

| Field             | Type   | Optional | Default | Description                                            |
| ----------------- | ------ | -------- | ------- | ------------------------------------------------------ |
| name              | string | yes      |         |                                                        |
| parent_account_id | string | yes      |         |                                                        |
| properties        | object | yes      |         | arbitrary properties accessible from the template      |
| user_profile      | object | yes      |         | a user profile object merged with user profile on send |

**Example:**

```js
POST https://api.courier.com/accounts/ACME

BODY
{
  name: "ACME Inc PRO MAX ULTRA",
  user_profile: {
    locale: "fr",
  }
}
```
