# Get an Account

**URL:** `https://api.courier.com/accounts/:account_id`

**Method:** `GET`

**Path Parameters:**

<Params>
  <Param name="account_id" type="string" required>

Id of the account to be returned.

  </Param>
</Params>

**Response Body:**

| Field               | Type   | Description                                            |
| ------------------- | ------ | ------------------------------------------------------ |
| id                  | string | Id of the account                                      |
| name                | string |                                                        |
| parent_account_id   | string |                                                        |
| default_preferences | object | Account level default_preferences object               |
| properties          | object | Arbitrary properties accessible from the template      |
| user_profile        | object | A user profile object merged with user profile on send |
| brand_id            | string | Brand associated with the account                      |

**Example:**

```js
// GET https://api.courier.com/accounts/ACME

// RESPONSE BODY
{
  id: "ACME",
  name: "ACME Inc",
  properties: {
    ceo: "Person",
  },
  default_preferences: {
    items: [{
      status: "OPTED_IN",
      id: "my-topic",
      type: "subscription_topic"
    }]
  },
  user_profile: {
    locale: "en-US",
    "slack": {
      "access_token": "xoxb-bunch-o-letters",
    }
  }
}
```
