# Get an Account

**URL:** `https://api.courier.com/accounts/:account_id`

**Method:** `GET`

**Path Parameters:**

| Key        | Type   |
| ---------- | ------ |
| account_id | string |

**Response Body:**

| Field             | Type   | Description                                            |
| ----------------- | ------ | ------------------------------------------------------ |
| id                | string | Id of the account                                      |
| name              | string |                                                        |
| parent_account_id | string |                                                        |
| properties        | object | Arbitrary properties accessible from the template      |
| user_profile      | object | A user profile object merged with user profile on send |

**Example:**

```js
// GET https://api.courier.com/accounts/ACME

// RESPONSE BODY
{
  id: "ACME",
  name: "ACME Inc",
  properties: {
    ceo: "Person",
  },
  user_profile: {
    locale: "en-US",
    "slack": {
      "access_token": "xoxb-bunch-o-letters",
    }
  }
}
```
