# Automations API

Courier's Automations API provides endpoints to:

- Invoke an existing automation template
- Invoke an adhoc automation (An automation supplied in the call via JSON)
- Get information about an automation run, the "Automation Run Object:

This section of the docs are limited to the Automations API. To learn more about Courier Automations,
including examples, instructions on how to use the designer, and information about what Automations
can do; Visit the automations reference available [here](https://www.courier.com/docs/automations/).
