# Get a Tenant

**URL:** `https://api.courier.com/tenants/:tenant_id`

**Method:** `GET`

**Path Parameters:**

<Params>
  <Param name="tenant_id" type="string" required>
    Id of the tenant to be returned.
  </Param>
</Params>

**Response Body:**

<TenantResponse />

**Example:**

```js
// GET https://api.courier.com/tenant/ACME

// RESPONSE BODY
{
  id: "ACME",
  name: "ACME Inc",
  properties: {
    ceo: "Person",
  },
  default_preferences: {
    items: [{
      status: "OPTED_IN",
      id: "my-topic",
      type: "subscription_topic"
    }]
  },
  user_profile: {
    locale: "en-US",
    "slack": {
      "access_token": "xoxb-bunch-o-letters",
    }
  }
}
```
