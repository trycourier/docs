## Usage

Messages that are sent to a list or audience can be archived in bulk using this API with the returned request_id.
You most likely would not need to use this API unless you're specifically building around this use-case. Reach out to Courier Support if you have any questions.

## Example

Method: `PUT`

URL: `https://api.courier.com/requests/1-62bbd4dc-d0a17ebd6ec04e7bbab5de60/archive`
