# Datadog

Courier supports Datadog as an observability integration. You can follow the steps below to setup the Datadog integration.

### Setup

- Create a Datadog account if you don't have one.
- Create a Datadog [API key](https://app.datadoghq.com/organization-settings/api-keys).
- Login to Courier and navigate to the Channels page.
- Click on Datadog channel and enter the API key from step 2.

### Datadog Site Parameter

Customers have the option to include a [site parameter](https://docs.datadoghq.com/getting_started/site/) to your Datadog integration.
