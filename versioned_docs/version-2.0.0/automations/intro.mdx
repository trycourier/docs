# Introduction

Automations is a fully managed orchestration service that makes it easy to design complex notification workflows with one API.

## The Courier Automation Toolset

How can you implement Automations?
* Using the `/automation/invoke` API
* Using the visual template designer

What can you build with Automations?
* Delay the delivery of a notification
* Fetch the latest data to include with a notification
* Cancel the delivery of a notification
* Schedule a reoccurring notification
