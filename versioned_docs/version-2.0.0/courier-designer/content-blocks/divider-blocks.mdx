## Channel Specific Options and Behaviors

### Email

In email notifications, you can select a color for your divider block, using the color picker, a hex value, or preset brand colors.

<Image
  img={require("../../assets/guides/divider-block/divider-block-color.png")}
  alt="Divider Block Color Picker"
/>

### SMS / Direct Message

In channels other than email, the divider block appears as three dashes or an uneditable divider.

<Image
  img={require("../../assets/guides/divider-block/divider-block-sms.png")}
  alt="SMS Divider"
/>

## Filtering a Divider Block 

Like with any content block, Diver Blocks can be hidden using a [conditions filter](../notification-settings/send-conditions.mdx/#for-content-blocks).
