# Pusher Beams

## Introduction

Pusher Beams can be used to send to either the users or to the interests. At least one `mode` out of web, apns or fcm is required.

## Send to interests

```json title=JSON
{
  "message": {
    "to": {
      "pusherBeams": {
        "interests": ["interest-1", "interest-2"],
        "mode": ["apns", "fcm"]
      }
    }
  }
}
```

## Send to users

```json title=JSON
{
  "message": {
    "to": {
      "pusherBeams": {
        "userIds": ["user-id-1", "user-id-2"],
        "mode": ["apns", "fcm"]
      }
    }
  }
}
```
